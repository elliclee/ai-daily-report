头条新闻

X热议Anthropic Claude Cowork引发"SaaS末日"恐慌，全球软件股市值蒸发约3000亿美元

Anthropic发布Claude Cowork插件系统，可自动化法律审查、销售营销、财务工作流等核心企业任务。这直接冲击了传统SaaS商业模式，引发全球软件股抛售潮——伦敦证交所集团股价两日暴跌约20%，印度IT股集体下挫（Infosys跌7.3%、TCS跌6%），多家投行将此事件标记为"SaaSpocalypse"（SaaS末日）。与聊天机器人不同，Claude Cowork能在企业系统内执行端到端的完整专业工作流。

via X @StratyQ、X @TheBean_media、X @nucleusnewss、X @grok

X热议Anthropic与OpenAI超级碗广告大战升级

Anthropic在超级碗期间投放广告，讽刺OpenAI和ChatGPT会在对话中插入侵入式广告，同时宣传Claude将保持无广告体验。Sam Altman回应称广告很有趣，但指责其"不诚实"——OpenAI最重要的广告原则正是不会这样做。Altman强调OpenAI致力于让AI免费惠及更多人，ChatGPT的免费用户已超过Claude的总用户数；同时批评Anthropic针对富人提供昂贵产品，并试图控制AI使用（如封锁竞争对手）。OpenAI的广告则主打Codex编程助手，已有50万次下载。

via X @grok、X @sama、X @devua_official

X热议NVIDIA即将完成对OpenAI 200亿美元投资

NVIDIA接近完成对OpenAI的200亿美元投资，这将是NVIDIA史上最大单笔投资，但远低于此前传闻的1000亿美元。黄仁勋澄清称，2025年9月宣布的"最高1000亿美元滚动投资"并非具有约束力的承诺。OpenAI本轮融资目标超过1000亿美元，估值已达8300亿美元，Amazon和SoftBank也参与投资。这笔交易将深化芯片巨头与AI领军者的绑定关系。

via X @SGUOIS、X @SwissCognitive、X @TheFinancialD、X @grok

AI驱动的裁员潮蔓延，Pinterest和Dow明确归因于AI转型

越来越多的公司将裁员归因于AI转型。Pinterest和Dow上个月宣布裁员时，明确提到部分原因是转向人工智能。这一趋势标志着企业正加大对AI的投资，同时精简人力资源，AI替代效应从理论走向现实。CBS News报道指出，AI正在从"辅助工具"转变为"人力替代者"。

via CBS News

X热议"这不是抛售，是轮动"——AI时代的资本重新配置

X用户@0xCaptainLevi分析指出，近两日约1万亿美元从软件和服务股中蒸发，但这不是全面抛售，而是资本轮动。资金正从AI威胁的业务流向AI无法触及的领域——制药、能源、国防、工业。纳斯达克跌1.5%、标普跌0.5%，但道指接近历史新高；欧洲指数创历史新高。Palantir跌11%，AMD跌17%（尽管CEO称需求"火热"）。与此同时，Eli Lilly因业绩超预期飙升10%，市值重回1万亿美元以上。"AI提升所有船只"的投资论已死，投资者正在区分"被AI增强"和"被AI颠覆"的企业。

via X @0xCaptainLevi
